# memoApp

### 使用手順
1. リポジトリをclone

    ```git clone https://github.com/doitumura/memoApp.git```

1. cloneしたリポジトリに移動

    ```cd memoApp```

1. branchをdevに変更

    ```git checkout dev ```

1. gemをインストール

    ```bundle install```

1. アプリを起動

    ```ruby app.rb```
    
1. ブラウザのアドレスバーからlocalhost:4567にアクセス
